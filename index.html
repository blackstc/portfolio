Tetris Clone

As a front-end project, I re-created the classic game Tetris. I made this game using only vanilla javascript and canvas.

Github Live Demo

Angular Translator

This was my first project using Angular. This is a simple app that uses the Bing translator API. The user can create a profile and then practice or take a 10 question challenge to test their translation skills. Their performance is recorded in a mongo database.

Github Live Demo

Cyber Tanks

Built in under 3 weeks and entirely pair-programmed, Cyber Tanks is a multiplayer browser game that integrates Phaser, Express, Mongo, and Socket.io. The aim was to make an exciting and 'arcadey' game with a unique controller - your phone.

Github Live Demo

Steam Games

This is an angular app created to view and get additional information on the games the user owns on Steam. The app uses OpenId to login to the users Steam account, additionally we used the Giant Bomb API to get the user additional information about the game.
